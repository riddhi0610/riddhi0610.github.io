In addition, my Github portfolio includes projects where I am learning new languages such as JavaScript and OpenCV Python. Below are some of the projects I have done over the course of 3-4 years!

Interactive ASL Recognition

A program that takes in the hand gestures displayed by the user and translates ASL. Utilized the framework mediapipe and the OpenCV library. Backend modules are palm detection and landmark. IDE is PyCharm and the language is Python.

Python, PyCharm, OpenCV

Controlling Volume by Hand Gestures

A program that allows the user to control the volume of their device with gestures involving their thumb and index finger. Utilized the framework mediapipe and the OpenCV library. Backend modules are palm detection and landmark. IDE is PyCharm and the language is Python.

Python, PyCharm, OpenCV
